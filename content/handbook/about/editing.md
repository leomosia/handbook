## Editing in GitHub

Use the page source link to find the file, then open a pull request with your
change.

## Editing Locally

```bash
npm install
npm run dev
```

Then edit Markdown under `content/handbook/`.

## Moving Content

Move content in a dedicated pull request. Rewrite or expand the content in a
separate follow-up pull request.

This keeps review clean and preserves useful file history.
